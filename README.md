# Bubble Mania 🫧

A fast-paced browser reflex game — pop bubbles, dodge bombs, and beat your high score.

## How to Play

- **Bubbles** appear randomly and vanish after a short time — click them to score points
- **Bombs** also appear — avoid clicking them or you lose a bomb life
- Miss **10 bubbles** or click **3 bombs** and it's Game Over
- Speed increases smoothly as your score climbs
- Bombs appear more frequently at higher scores

## Features

- Smooth difficulty scaling — speed increases 1.8% per point scored
- Score-based level system (LVL 1 → 2 → 3) with confetti bursts
- Bomb lives indicator in the HUD (💣 💣 💣)
- Miss health bar that turns red as you approach game over
- Web Audio API sound effects — pop, miss, explosion, level up, game over
- Top 5 high scores saved locally
- Single self-contained HTML file — no frameworks, no dependencies

## Play

Just open `reflex-circles.html` in any browser — no server needed.
